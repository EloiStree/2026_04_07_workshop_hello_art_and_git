## Workshop Arts And Git 😁

Le but de la formation est de vous former à la XR (Virtual Reality, Augmented Reality et Mixed Reality) dans un groupe composé de 12 développeurs et 12 artistes.

Pour les développeurs, Git est une seconde nature.  
Pour les artistes, Git est souvent un vrai film d’horreur xD.

Cette semaine est là pour vous montrer que ce n’est pas si terrible, et que c’est même l’outil le plus utile de toute votre vie.  
C’est le premier outil informatique que l’on devrait apprendre aux enfants à l’école primaire 😉  
_(Après un logiciel de dactylographie ⌨️ et Krita 💘)_

### Pourquoi apprendre Git

- Ne jamais perdre ses projets (backup parfait)
- Travailler en équipe sur le même projet
- Travailler sur différents ordinateurs sans problème
- Créer des boîtes à outils réutilisables
- …

Ici, nous sommes dans une formation XR. Durant la formation, je parie que vous aurez envie de voir vos créations 3D directement dans les casques de VR.  
Sinon, je ne sais pas ce que vous faites dans cette formation XR 😜😅

Comme vous avez un cours plus axé sur la création que sur les Game Engines pour les artistes, il faudra compter sur les développeurs pour visualiser vos créations en XR.  
(Je vous invite fortement à apprendre comment voir vos créations dans les game engines et à ne pas dépendre uniquement des développeurs.)

Le but, à la fin de la semaine, est que vous soyez à l’aise avec la phrase : **Add, Commit, Pull, Push** 🪄 sur une branche🪵

Je ne vois que les bases avec vous, mais c’est à vous de continuer à vous former tout au long de la formation, par la pratique et parfois par l’échec.

Git est un outil indispensable dans le jeu vidéo.  
Ne pas le maîtriser équivaut à limiter fortement ses chances d’emploi dans l’industrie.

Bonne apprentissage à tous !  
Allez, c’est parti.
